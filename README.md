# Motorola CP040 Firmware flashing tool

This project describes an adaptor to allow firmware flashing of the Motorola CP040 radio.
The adapter is designed to be fitted between a USB to Serial converter based programming cable and the radio.
It is intended for use with Motorola firmware update file such as:
https://www.radioscanner.ru/files/download/file19831/portable_firmware_kit_r03.07.01_with_codeplug_r10.00.zip

This will allow firmware flashing using a cheap eBay programming cable instead of an expensive Motorola RLN5583B programming cable and a RIB.
